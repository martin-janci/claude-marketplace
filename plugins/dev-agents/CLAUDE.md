# Dev Agents

Core development agents and skills for backend, database, security, debugging, and refactoring.

## Agents

| Agent | Description |
|-------|-------------|
| `backend-developer` | Backend development specialist |
| `rust-engineer` | Rust systems programming expert |
| `debugger` | Systematic debugging specialist |
| `refactoring-specialist` | Code refactoring expert |
| `database-optimizer` | Database optimization (PostgreSQL) |
| `postgres-pro` | PostgreSQL expertise |
| `api-designer` | API specification and design |
| `dependency-manager` | Dependency management |
| `devops-engineer` | DevOps and infrastructure |
| `security-engineer` | Security and compliance |
| `chaos-engineer` | Chaos engineering and resilience testing |

## Skills

- **test-driven-development/** - TDD workflow (Red-Green-Refactor)
- **systematic-debugging/** - Methodical debugging approach
- **verification-before-completion/** - Verification patterns
- **testing-anti-patterns/** - Testing pitfalls to avoid
- **writing-skills/** - Skill authoring guide
- **using-git-worktrees/** - Git worktree management for parallel work

## Commands

- `/deps-check` - Check and update dependencies
- `/clean` - Clean build artifacts
- `/branches` - Branch management

## Usage

Spawn agents for development tasks:
```
/spawn backend-developer "implement user authentication"
/spawn debugger "investigate the login timeout issue"
/spawn database-optimizer "optimize slow queries in users table"
```
